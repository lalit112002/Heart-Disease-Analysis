# Heart-Disease-Analysis
Analyzed a heart disease dataset to explore relationships among various medical attributes such as age, cholesterol, blood pressure, and heart rate. Applied data cleaning, feature correlation, and visualization techniques to identify the most significant factors contributing to heart disease risk.
Using Python libraries (Pandas, Matplotlib, Seaborn) to uncover patterns and correlations between patient attributes and heart disease risk. Derived meaningful insights through data visualization and statistical analysis.
